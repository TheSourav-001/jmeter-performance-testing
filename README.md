# jmeter-performance-testing
A structured performance and load testing project using Apache JMeter, demonstrating Thread Groups, HTTP Samplers, Assertions, Loop Controllers, Timers, and result analysis through listeners.

### Folder Details
- **test-plan/**: Contains JMeter test plans (`.jmx` files) for performance testing.
- **report/**: Contains performance test reports in PDF format.
- **screenshots/**: Screenshots of JMeter test setup and results.
- **README.md**: Project overview and instructions.

## Key Features
- Load testing using JMeter.
- Thread group, timers, and assertions configured.
- Performance analysis reports generated.
- Visual demonstration through screenshots.

## How to Use
1. Open the `.jmx` file in Apache JMeter.
2. Run the test plan.
3. Review the generated results and screenshots in the `report/` and `screenshots/` folders.

## Notes
- Ensure Apache JMeter is installed before running the test plan.
- This project is designed for educational and portfolio purposes.
